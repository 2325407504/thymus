Basic Usage 

<head th:substituteby="frags/head :: header">
<meta charset="utf-8" />
<!-- Thymus script -->
<script th:remove="all" src="js/thymus.js" id="thymus" 
	data-context-path="."
	data-head-frag="th:substituteby=frags/head :: header"
	data-onfragsloaded="init(this)"></script>
</head>

Thymeleaf Integration

Thymeleaf is a great server-side template engine for Java and is the inspiration behind Thymus. In fact, by default Thymus uses the same default HTML5 attributes for includes/substitutions and fragment definitions. Similar
